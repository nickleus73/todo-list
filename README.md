# Install Todo-list
Required : install of node js

`sudo npm install`

# Create a todo
`todo-list -t "My TODO"`

# Read a todo-list
`todo-list -r`

Display : 

`#1 - First todo
\#2 - Second todo
\#3 - Other todo`

# Deltet a todo-list
Example to delete the todo "#3 - Other todo", enter

`todo-list -d 3`
